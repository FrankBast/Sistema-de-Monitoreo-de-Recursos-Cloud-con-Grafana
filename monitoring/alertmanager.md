# Alertmanager – Gestión de alertas

## Principios

- Alertas basadas en impacto
- Separación por severidad
- Ruteo por rol (ops, soc)

## Ejemplos de alertas

- CPU alta sostenida
- Disco crítico
- Servicio web caído
- Disponibilidad mensual baja

Cada alerta incluye:
- summary
- description
- runbook_url
