# Sistema-de-Monitoreo-de-Recursos-Cloud-con-Grafana

Repositorio académico para el diseño e implementación de un sistema
de monitoreo basado en Prometheus y Grafana.

## Estructura

- docs/: documentación funcional y de diseño
- monitoring/: lógica de monitoreo y alertas
- dashboards/: dashboards Grafana
- infra/: contexto y etiquetado
- tests/: validaciones y pruebas
